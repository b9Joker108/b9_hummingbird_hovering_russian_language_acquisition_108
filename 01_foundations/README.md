# Foundations: A1-A2 Level

*Начало пути - The Beginning of the Path*

## Overview

This module provides the essential foundations for Russian language acquisition, starting from absolute beginner level (tabula rasa) to basic competence.

## Learning Objectives

By completing this module, you will:
- Master the Cyrillic alphabet
- Pronounce Russian sounds correctly
- Use basic grammar structures
- Command 500-1000 essential words
- Conduct simple conversations
- Understand basic cultural contexts

## Recommended Timeline

- **Week 1-2**: Cyrillic alphabet and basic phonetics
- **Week 3-4**: Essential grammar patterns
- **Week 5-6**: Core vocabulary building
- **Week 7-8**: Simple conversations and cultural context
- **Week 9-12**: Integration and practice

## Study Tips

1. **Daily Practice**: 30-60 minutes minimum
2. **Spaced Repetition**: Use flashcard apps (Anki recommended)
3. **Immersion**: Listen to simple Russian audio daily
4. **Writing**: Practice Cyrillic handwriting
5. **Speaking**: Record yourself, compare to native speakers

## Orthodox Christian Context

Even at beginner level, you'll encounter:
- Basic liturgical greetings (*Христос воскресе!* - Christ is risen!)
- Common prayers (*Отче наш* - Our Father)
- Calendar references (Church calendar basics)
