# Practice & Exercises

*Практика и упражнения*

## Overview

This module provides structured practice activities across all skill areas and proficiency levels. Consistent, varied practice is essential for language acquisition and cultural literacy development.

## Practice Philosophy

### Integrated Skills Approach

**Four Skills Balanced**:
1. **Reading** (Чтение) - Input, vocabulary, grammar
2. **Listening** (Слушание) - Comprehension, pronunciation models
3. **Speaking** (Говорение) - Production, fluency, confidence
4. **Writing** (Письмо) - Accuracy, expression, organization

**Cultural Integration**: Practice embedded in Orthodox and Russian cultural contexts

### Progressive Difficulty

**Spiral Curriculum**:
- Return to topics with increased complexity
- Build on previous knowledge
- Connect across modules
- Integrate new with known

### Spaced Repetition

**Optimize Memory Consolidation**:
- Review at optimal intervals
- Active recall strengthens retention
- Use flashcard apps (Anki recommended)
- Regular revisiting of material

## Practice Structure by Level

### Beginner (A1-A2)

#### **Daily Practice Routine** (60-90 min)

**Morning (30 min)**:
1. **Alphabet Review** (5 min)
   - Cyrillic recognition
   - Handwriting practice
   - See `/11_practice/alphabet/`

2. **Phonetics Drill** (10 min)
   - Pronunciation practice
   - Minimal pairs
   - Recording yourself
   - See `/11_practice/pronunciation/`

3. **Vocabulary Review** (15 min)
   - Anki flashcards
   - 20-30 new words per week
   - Review previous words

**Afternoon (30-45 min)**:
1. **Grammar Study** (20 min)
   - One concept per day
   - Exercises
   - Examples
   - See `/11_practice/grammar/beginner/`

2. **Reading Practice** (10-15 min)
   - Simple texts
   - Dialogue practice
   - Beginner readers
   - See `/11_practice/reading/beginner/`

3. **Listening** (10 min)
   - Beginner podcasts
   - Simple videos
   - Repeat-after exercises

**Evening (15 min)**:
1. **Speaking Practice** (10 min)
   - Read aloud
   - Record yourself
   - Practice dialogues
   - Orthodox prayers aloud

2. **Writing** (5 min)
   - Copy sentences
   - Simple diary entry
   - Handwriting practice

## Practice Tips

### Consistency Over Intensity

- **Daily practice** better than long occasional sessions
- **15 minutes daily** > 2 hours weekly
- Build habits
- Sustainable routine

### Active Recall

- Test yourself
- Don't just re-read
- Use flashcards effectively
- Spaced repetition

### Multimodal Learning

- Read, listen, speak, write
- Use visual aids
- Incorporate cultural activities
- Vary practice types

### Immersion Techniques

- Think in Russian
- Label environment
- Russian-only time blocks
- Cultural participation

### Error Tolerance

- Mistakes are learning opportunities
- Don't fear speaking
- Self-correction develops
- Progressive improvement
